## Setup

```
git clone https://github.com/EvaBro/faces_paradigm
```

### Stimuli
The stimuli are available upon request.

### Dependencies
This paradigm requires [stim_utils](https://github.com/EvaBro/stim_utils). Clone it as a sibling folder:

```
git clone https://github.com/EvaBro/stim_utils
```

### Final folder structure
```
YourFolder/
├── stim_utils/
└── faces_paradigm/
    └── Faces_original
    └── Faces_scrambled
    └── Target_images
    └── Instructions
```
